## 案例 1：二手数码 · 外部引流

**业务**：卖家想引导买家到微信看更多实拍图。

**文案**：
```
iPhone15 国行在保，细节图不够？加薇-信 abc123 发你更多实拍。
```

**代码**：

```python
from check_text import check

result = check("iPhone15 国行在保，细节图不够？加薇-信 abc123 发你更多实拍。")
print(result["verdict"])  # 不建议发布
```

**改写建议**：细节图已全部上传，有需要平台内私信沟通。

## 案例 11：店铺运营 · 批量上架

**业务**：运营上架前批量扫描多条商品。

**代码**：

```python
from business_cases import batch_check

listings = [
    {"id": "SKU001", "title": "九成新 iPad", "desc": "个人自用，走平台交易。"},
    {"id": "SKU002", "title": "考研资料合集", "desc": "肖秀荣考研政治PDF电子版。"},
    {"id": "SKU003", "title": "机械键盘", "desc": "加微信看细节，13800138000。"},
]
for row in batch_check(listings):
    print(row["id"], row["check"]["verdict"])
# SKU001 通过 | SKU002 不建议发布 | SKU003 不建议发布
```

## 接入自己的业务

```python
import sys
from pathlib import Path

sys.path.insert(0, str(Path("scripts")))  # 指向 skill/scripts 目录
from check_text import check

def before_publish(title: str, desc: str) -> bool:
    r = check(f"{title} {desc}")
    if r["verdict"] != "通过":
        print(f"拦截: {r['hits']}")
        return False
    return True
```
